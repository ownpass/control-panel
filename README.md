## Installation

1. Clone the reposotory
2. $ `cd control-panel`
3. Make sure [Node v4+](https://nodejs.org/en/download/) & [npm v3](https://docs.npmjs.com/getting-started/installing-node) are installed
4. $ `npm install`
5. $ `npm run typings install`
6. $ `npm start`
